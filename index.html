Yet Another Cheatsheet Git

Git (/ɡɪt/)[8] is free and open source software for distributed version control: tracking changes in any set of files, usually used for coordinating work among programmers collaboratively developing source code during software development. Its goals include speed, data integrity, and support for distributed, non-linear workflows (thousands of parallel branches running on different systems)

Installation

Linux

Most if not all Linux distros come with git but if you wanted to install it on Debian based distros try 'sudo apt-get install git' on the command line

Mac

For information on installing git on Mac check out GitHub for Mac

Windows

For information on installing git on Windows check out GitHub for Windows

Setup

git config --global user.name “[firstname lastname]”
Set your name for credit when reviewing version history

git config --global user.email “[valid-email]”
Set an email address to be associatied with each history marker

Initiation

git init
Initialise an existing directory as a Git repository

git clone [url]
retrieve a repository from a hosted location
